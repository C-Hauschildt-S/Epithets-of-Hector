This is a project for my Master's thesis on the epithets of Hector in Homer's _Iliad_. 
It combines CLTK and DICES to map all epithets used for Hector along with their morphosyntax and cooccurrence with other epithets.
It relies on both standard XML-files and Treebanked files of the _Iliad_.
The current code is also available as a Streamlit app: https://epithets-of-hector.streamlit.app/
I am currently working on incorporating scansion to further find metrically equivalent epithets.
